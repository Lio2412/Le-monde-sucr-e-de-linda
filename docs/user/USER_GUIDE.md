# Guide d'Utilisation - Le Monde Sucré de Linda

## 👋 Introduction

Bienvenue dans le guide d'utilisation du dashboard administrateur du Monde Sucré de Linda. Ce guide vous aidera à comprendre et utiliser toutes les fonctionnalités disponibles.

## 🔑 Accès au Dashboard

### Connexion
1. Accédez à `/admin/login`
2. Entrez vos identifiants administrateur
3. Cliquez sur "Se connecter"

### Sécurité
- Changez régulièrement votre mot de passe
- Déconnectez-vous après chaque session
- N'utilisez pas de connexion publique

## 📝 Gestion du Contenu

### Recettes

#### Création d'une Recette
1. Cliquez sur "Recettes" dans le menu
2. Sélectionnez "Nouvelle Recette"
3. Remplissez les champs :
   - Titre
   - Description
   - Ingrédients
   - Instructions
   - Images
4. Utilisez l'éditeur visuel pour le formatage
5. Prévisualisez avant publication
6. Cliquez sur "Publier"

#### Modification d'une Recette
1. Trouvez la recette dans la liste
2. Cliquez sur "Modifier"
3. Effectuez vos changements
4. Sauvegardez ou prévisualisez

### Articles de Blog

#### Publication d'Articles
1. Accédez à la section "Blog"
2. Cliquez sur "Nouvel Article"
3. Rédigez votre contenu
4. Ajoutez des médias si nécessaire
5. Définissez les métadonnées SEO
6. Planifiez la publication si souhaité

## 🖼️ Gestion des Médias

### Bibliothèque de Médias
- Accédez via "Médias" dans le menu
- Organisez par dossiers
- Utilisez la recherche pour retrouver les fichiers
- Prévisualisez avant utilisation

### Upload d'Images
1. Cliquez sur "Ajouter"
2. Sélectionnez ou glissez-déposez
3. Attendez le traitement automatique
4. Ajoutez des descriptions ALT

## 💬 Modération

### Commentaires

#### Validation des Commentaires
1. Accédez à "Modération > Commentaires"
2. Examinez les commentaires en attente
3. Approuvez ou rejetez
4. Ajoutez une note si nécessaire

#### Réponses aux Commentaires
1. Trouvez le commentaire
2. Cliquez sur "Répondre"
3. Rédigez votre réponse
4. Publiez
